# 🛍️ Sistema Web de Perfumería

Aplicación web desarrollada en Python orientada a la gestión de productos de perfumería, con funcionalidades de carrito de compras y manejo de datos mediante APIs.

## 🚀 Descripción

Este proyecto consiste en el desarrollo de una aplicación web que permite visualizar productos, agregarlos a un carrito de compras y gestionar la información a través de una base de datos.

Se implementaron conceptos de programación orientada a objetos, integración con APIs REST y diseño de interfaces para mejorar la experiencia del usuario.

## 🛠️ Tecnologías utilizadas

- Python  
- NiceGUI  
- Supabase  
- APIs REST  
- SQL  
- HTML / CSS
